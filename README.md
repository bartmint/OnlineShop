# General Info
- This is Asp net core online shop using repository-service pattern.
## Technologies
- ASP NET CORE
- MVC,
- ENTITY FRAMEWORK CORE,
- LINQ,
- SQL SERVER,
- DATA ANNOTATIONS,
- IMAPPER
- BOOTSTRAP
## Features
- register/login user
- buy/add/remove products from cart
- CRUD operations by admin
- filters/paginate/sort products
